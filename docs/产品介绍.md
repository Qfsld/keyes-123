# 1. 产品介绍

**Keyes brick 24合一传感器套装**

![e10d0036e08e6015cefaa7fe9a4e0bf6](./media/e10d0036e08e6015cefaa7fe9a4e0bf6.jpeg)

## 1.1 简介

keyes brick 24合一传感器套装主要包含了我们常用的24款传感器/模块，
还有对应的keyes UNO R3开发板、传感器扩展板和连接线。24款传感器/模块上都带有防反接口，
和我们提供的传感器扩展板接口完全匹配。使用时，我们只需要将传感器扩展板堆叠在keyes UNO R3开发板，
利用1根自带的连接线将传感器/模块连接在扩展板上，简单方便。
